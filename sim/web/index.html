About & future plans

This dashboard streams a cycle-accurate view of the tiny-gpu RTL simulation (cocotb + Icarus Verilog) over a WebSocket, or replays a saved .jsonl trace. Every value shown - core/fetcher states, per-thread PCs/registers/active-mask, i-cache hit/miss, and the full data memory - is read directly from simulator signals each cycle, not reconstructed after the fact.

Planned desktop version: a native (PyQt/Qt) build reusing the same JSON snapshot protocol and test/helpers/snapshot.py extraction layer, adding waveform-style scrubbing across the whole run, multi-kernel side-by-side comparison, and export to VCD/GTKWave for deeper signal debugging. See CHANGELOG.md for status.
